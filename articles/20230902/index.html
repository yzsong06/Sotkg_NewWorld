HTML 语法笔记

标题


                        <h1>标题1</h1>
                        <h2>标题2</h2>
                        <h3>标题3</h3>
                        <h4>标题4</h4>
                        <h5>标题5</h5>
                        <h6>标题6</h6>

字体样式


                        <em>强调</em>
                        <i>斜体</i>
                        <b>粗体</b>
                        <strong>加重</strong>
                        <small>小号字体</small>
                        <strike>删除线</strike>

输入框

<input type="text" /> 定义单行文本输入框
                        <input type="password" /> 定义密码输入框
                        <input type="checkbox" /> 定义复选框
                        <input type="color" /> 用于包含颜色的输入字段
                        <input type="date" /> 输入日期的字段
                        <input type="email" /> 邮件地址类型的输入字段
                        <input type="file" /> 定义文件选择字段和浏览按钮
                        <input type="image" /> 上传图片
                        <input type="month" /> 选择月份和年份
                        <input type="number" /> 定义数字输入字段
                        <input type="radio" /> 定义单选按钮
                        <input type="range" /> 滑动条控件
                        <input type="reset" /> 定义重置按钮，将所有表单值重置为默认值
                        <input type="search" /> 定义搜索字段
                        <input type="submit" /> 定义提交表单数据的按钮
                        <input type="tel" /> 定义电话号码字段
                        <input type="time" /> 选择时间（不包含时区）
                        <input type="url" /> URL地址类型的输入字段
                        <input type="week" /> 选择周和年份

表单

*注意：name 和 id 可以用于这些标签，以便唯一访问它们
                        <form action="" method="" target=""></form> 创建 HTML 表单的容器
                        <input type="" /> 定义输入控件，根据 type 属性不同而不同
                        <label for=""></label> 定义表单元素的标签，与 for 属性相对应
                        <select></select> 定义表单中的下拉列表
                        <option value=""></option> 定义下拉列表中的选项
                        <textarea rows="" cols=""></textarea> 定义多行输入字段
                        <fieldset></fieldset> 在表单中对相关元素进行分组
                        <legend></legend> 为 fieldset 标签定义标题
                        <datalist><option value="">..</option></datalist> 为输入标签指定预定义选项列表
                        <output></output> 用于表示计算结果（要使用这个标签，需要在 form 标签内使用 oninput 属性）
                        <optgroup></optgroup> 用于分组相关选项

换行符

<br />

图片

如果图片存储在您的工作空间中，可以像这样添加图片：
                        <img src="img_girl.jpg" width="400" height="400" />
                        或者您可以直接从浏览器添加图片，像这样：
                        <img
                          src="https://cdn.mos.cms.futurecdn.net/hFm4iWXhbw4c4rdcMH8tUD.jpg"
                          width="400"
                          height="400"
                        />

HTML头部

<head></head> 元数据的容器
                        <title></title> 项目的标题
                        <link rel="stylesheet" href="" /> 链接到外部样式表
                        <meta /> 指定字符集、页面描述、关键字、文档作者和视口设置
                        <style></style> 用于为单个文档定义样式信息
                        <base href="" /> 指定所有相对 URL 的基本 URL 和/或目标
                        <script></script> 定义客户端 JavaScript

水平分隔线

<hr />

锚点

<a href="url">链接文本</a> 链接到 url
                        <a href="#yourSection">链接文本</a> 链接到页面中的某一部分
                        <a href="url" target="_blank">链接文本</a> 在新标签页中打开链接

表格

<table></table> 定义表格
                        <tr></tr> 定义表格行
                        <th></th> 定义表格标题
                        <td></td> 定义表格数据

列表

<ul></ul> 定义无序列表
                        <ol></ol> 定义有序列表
                        <li></li> 定义列表项
                        <dl></dl> 定义描述列表
                        <dt></dt> 定义描述列表中的术语
                        <dd></dd> 描述描述列表中的术语

按钮

<button></button> 定义可点击的按钮

SVG

<svg></svg> SVG 图形的容器

视频标签

<video controls src="movie.mp4" type="video/mp4">
                          该浏览器不支持该视频格式！
                        </video>

无序列表

                          <li>项目1</li>
                          <li>项目2</li>
                          <li>项目3</li>
                        </ul>

有序列表

<ol>
                          <li>项目1</li>
                          <li>项目2</li>
                          <li>项目3</li>
                        </ol>

HTML语义化

<header></header> 指定文档或部分的标题
                        <main></main> 指定文档的主要内容
                        <footer></footer> 定义文档或部分的页脚
                        <article></article> 定义独立、自包含的内容
                        <aside></aside> 定义与页面内容无关的内容
                        <figure></figure> 指定独立的内容
                        <nav></nav> 定义导航链接
                        <section></section> 定义文档中的一个部分

语义化媒体标签

<figure></figure> 嵌入带注释的图片、插图、照片、代码等。
                        <figcaption></figcaption> 为 <figure> 添加标题/注释。
                        <picture></picture> 响应式图像插入，允许开发人员为不同的上下文提供不同的图像。
                        
                        <video poster=" " autoplay loop controls ></video>嵌入视频到网页中。
                           poster 是视频播放前显示的图像路径。
                           autoplay 自动播放视频。
                           loop 触发视频是否重复播放。
                           controls 显示或隐藏浏览器的播放器按钮。
                        
                        <audio></audio> 将音频嵌入到网页中。
                        <source></source> 必须在 <picture>、<video> 或 <audio> 内部定义不同版本的内容。

音频

<audio></audio> HTML <audio> 元素用于嵌入声音内容。

脚本

<canvas></canvas> 与 canvas 脚本 API 或 WebGL API 结合使用，用于绘制图形和动画。
                        <noscript></noscript> 定义一个 HTML 部分，如果页面上某个脚本类型不受支持或脚本当前处于关闭状态，则插入该部分。
